# cube

L IN GIU'

## primo strato

mettere il giusto sotto e fare l'algoritmo NON dalla faccia bianca

```
R  U  R' U'
```

## secondo strato

<- sinistra

```
U' L' U  L  U  F  U' F'
```

-> destra

```
 U  R  U' R' U' F' U  F
```

## terzo strato

per la croce

```
F  R  U  R' U' F'
```

per gli spigoli (U gira la gialla)

```
R  U  R' U  R  U2  R'
```

per L'ANGOLO GIUSTO (U gira la gialla, angolo "giusto a sx, "sbagliato" a dx)

