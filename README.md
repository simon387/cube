# cube

L IN GIU'

## primo strato

mettere il giusto sotto e fare l'algoritmo NON dalla faccia bianca

```
R  U  R' U'
```

## secondo strato

<- sinistra

```
U' L' U  L  U  F  U' F'
```

-> destra

```
 U  R  U' R' U' F' U  F
```

## terzo strato

```F  R  U  R' U' F'```

```R  U  R' U  R  U2  R'```

per L'ANGOLO GIUSTO

R  U' L' U  R' U' L  U
